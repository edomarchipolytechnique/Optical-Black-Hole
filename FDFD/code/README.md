# FDFD Code

Python scripts for building and plotting finite-difference frequency-domain optical black-hole simulations.

## Code

- `fdfd.py`: implements the 2D FDFD solver, PML boundaries, sources, annular permittivity, and Schwarzschild/Kerr-Newman simulation helpers.
- `plots_fdfd.py`: generates FDFD field, phase, annulus, geodesic, and Poynting-vector visualizations.
- `schwarzschild_fdfd.py`: entrypoint for generating Schwarzschild FDFD figures.
- `kerr_newman_fdfd.py`: entrypoint for generating Kerr-Newman FDFD figures.
