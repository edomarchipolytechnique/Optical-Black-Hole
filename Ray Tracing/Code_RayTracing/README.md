# Ray Tracing Code

Python scripts for continuous and annular Snell-law ray tracing in optical black-hole media.

## Code

- `ray_tracing_snell.py`: implements Snell-law ray propagation through annular refractive-index layers.
- `plots_ray_tracing_snell.py`: generates ray-tracing figures and error plots for Snell-law annular simulations.
- `error_annuli_snell.py`: entrypoint for studying how ray-tracing error changes with annulus count.
- `error_nb_snell.py`: entrypoint for studying ray-tracing error across impact-parameter or refractive-index cases.
- `ray_tracing_continuous_vary_n.py`: traces rays while varying refractive-index profile parameters.
- `ray_tracing_continuous_vary_b.py`: traces rays while varying impact parameter.
